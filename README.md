相比原项目，本工程是为了测试unity中的AssetBundle打包与读取资源，后又添加AI，现实了简单的人机对战。
### 改动一：添加AssetBundle
采用AssetBundle动态加载Prefabs资源。AssetBundle读取的是本地的打包文件。
### 改动二：整理代码
按照个人习惯整理代码、命名变量等，添加函数
### 改动三：采用Unity2D开发
本项目将原先的Texture显示方式，改为2D的Sprite显示
